# ATM System

This is an ATM system build in Node.js with TS.

## Run

To run this code you need to have installed [Node.js](https://nodejs.org/en/), and [Yarn](https://yarnpkg.com/).

Install the libraries:

```shell
yarn
# or
yarn install
```

Run in development mode:

```shell
yarn dev
# or
yarn run dev
```

Run project build:

```shell
yarn build
# or
yarn run build
```
